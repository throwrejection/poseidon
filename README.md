# Poseidon

Poseidon is a ERC-721 token.

```
# installing
sudo npm install -g truffle
sudo npm install -g ganache-cli
npm install
cp .env.sample .env && vim .env

# testing
ganache-cli
truffle test

# running
truffle migrate --reset --network dev
npm run dev
```
